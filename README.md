# Bookstore Management API

This is a Node.js project for managing a book database, built as part of the Full Stack Cloud Developer course.

## Files Structure
- `index.js`: Main server file containing all the endpoints.
- `booksdb.js`: Contains the mock data for books.
- `package.json`: Project dependencies and scripts.

## Setup Instructions
1. Run `npm install` to install Express.
2. Run `npm start` to start the server.
3. Use Postman to test the endpoints at `http://localhost:5000`.

## Implemented Features
- Callbacks, Promises, and Async/Await for data fetching.
- User registration and review management.
